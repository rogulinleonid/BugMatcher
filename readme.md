## BugMatcher

**Автоматизированная система проверки домашних заданий по курсу тестирования**

### О проекте

BugMatcher — это инструмент для преподавателей, который автоматизирует проверку студенческих работ по тестированию ПО. Система сопоставляет найденные студентами баги (зафиксированные в YouTrack) с эталонными данными из Google Sheets и формирует отчёт об успеваемости.

### Ключевые возможности

- **Интеграция с YouTrack**: автоматическое получение тикетов со найденными багами.
- **Работа с Google Sheets**: чтение эталонных данных о преднамеренно внесённых ошибках.
- **Интеллектуальное сопоставление**: сравнение описаний багов с использованием алгоритмов текстового анализа.
- **Формирование отчётов**: автоматическая генерация CSV‑отчётов с результатами проверки.
- **Конфигурируемость**: настройка через JSON‑файл без правки кода.

### Как это работает

1. Студенты заводят тикеты в YouTrack с описанием найденных багов.
2. BugMatcher выгружает эти тикеты через API.
3. Система получает эталонный список багов из Google Sheets.
4. Алгоритм сопоставляет найденные баги с эталонными по семантической близости.
5. Формируется отчёт с указанием:
   - студента;
   - номера тикета;
   - идентифицированного бага;
   - уровня уверенности сопоставления.

### Технические детали

- **Язык**: Python 3.x.
- **Зависимости**: `youtrack-python`, `google-api-python-client`, `pandas`, `scikit-learn`.
- **Формат отчёта**: CSV.
- **Конфигурация**: JSON‑файл с настройками доступа к сервисам.

### Зачем это нужно

- **Экономия времени**: автоматизация рутинной проверки заданий.
- **Объективность**: единые критерии оценки для всех студентов.
- **Прозрачность**: чёткая привязка найденных багов к эталонам.
- **Масштабируемость**: обработка сотен тикетов за минуты.

### Начало работы

1. Настройте доступ к YouTrack и Google Sheets (API‑токены, credentials).
2. Заполните эталонную таблицу в Google Sheets.
3. Отредактируйте `config.json` под свои параметры.
4. Запустите скрипт: `python main.py`.
5. Получите отчёт в формате CSV.
